REGISTRATION
First Name
  • Le prénom est obligatoire.
  • Le prénom doit contenir au moins 3 caractères.
 Last Name
  • Le nom est obligatoire.
  • Le nom doit contenir au moins 3 caractères.
 Address
  • L'adresse est obligatoire.
  • L'adresse doit contenir au moins 3 caractères.
 Email
  • L'email est obligatoire.
  • L'email est invalide.
  • L'email doit contenir un point.
 Password
  • Le mot de passe est obligatoire.
  • Le mot de passe doit contenir au moins 8 caractères.
  • Le mot de passe doit inclure une majuscule, une minuscule, un chiffre et un caractère spécial.
 Repeat Password
  • Les mots de passe ne correspondent pas.
  • Le repeate password est obligatoire.
  • Le repeate password doit contenir au moins 8 caractères.
  • Le repeate password doit inclure une majuscule, une minuscule, un chiffre et un caractère spécial.
Male Female Other
  • Le genre est obligatoire.
Date of Birth
  • La date de naissance est obligatoire.
